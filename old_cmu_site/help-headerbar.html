WEATHER WIDGET VERSION


Steps for adding the sticky header text promo bar shown on this page.


STEP 1.) Copy Files
Copy the "headerbar-promo" folder into your website folder. Files should stay inside this folder. Copy the "headerbar-promo.js" and "help-headerbar.html" into your website folder. Open the "help-headerbar.html" and go to the next step.

If you will be using the weather widget version copy over the "headerbar-WEATHER.js" and the "help-headerbar-WEATHER.html".


STEP 2.) Test
Open the "help-headerbar.html" (this page) in your website folder. You should see the header bar at the top of the page. If you scroll, the header bar should stay visible at the top of the webpage. If it's not working, re-do the step above.


Note: Use the "Global Options" below to add the bar to all your webpages without having to edit each webpage.


STEP 3.) Add </head> Css Code
To use the weather widget header bar go to help-headerbar-WEATHER.html

To use the header bar shown on this page, select, copy and paste the following line of code into the "head" of your existing webpage(s) that you want the header bar on. Put the code just before the </head> near the top of the page. Make backups of your webpages before you start.



Global Option: You can optionally add this globally to your global css file and skip the above step. Copy the code from the "headerbar-style.css" and paste it at the bottom of your global style .css file. Edit the following line adding the folder name for the background image:

background-image: url("headerbar-promo/hbar-background.png");

Or if your css files are inside a sub-folder edit to this:

background-image: url("../headerbar-promo/hbar-background.png");


STEP 4.) Add HTML Code

To use the header bar shown on this page, select, copy and paste the following line of code into your existing webpage(s). This code should go just after the <body> tag in your HTML page(s).



Global Option: You can optionally copy the code from the "headerbar-promo.js" and paste it into your "header.js" file. Make backups before you add this code.

STEP 5.) Test
Test the webpage you added the header bar to in a web browser. The bar should be visible and float down with the page as your scroll down on the page.


STEP 6.) Edit Icons & Links
Edit the "headerbar-promo.js" to edit the promo text and link, the social icon links and to add and remove icons.
  1. Edit the promo text and link on line #39 in the "headerbar-promo.js".
  2. Edit the social links in the headerbar-promo.js to link to your social accounts.
  3. Social icon images and the background flower image are in the "headerbar-promo" folder. You can replace these images with your own. Icons should be 30 x 30 pixels.
  4. Edit the "headerbar-promo.js" with the other image names to use other icons.
  5. The PSD file in the headerbar-promo folder is a Photoshop layered icon file.


OPTIONS
REMOVING / ADDING SOCIAL ICONS: | More details
Edit the "headerbar-promo.js". Each social icon group of code includes an identifying note and 3 lines of code. Delete the code group to remove any social icon.

Copy any group of 3 lines of code and paste it below itself to add another icon. Edit the image and links. Download more 30 x 30 icons.


COLORS & CSS OPTIONS: | More details
Edit the "headerbar-style.css" to change background colors, fonts, the background image and padding and margins.


ICONS HIDING ON MOBILE:
Some icons in the "headerbar-promo.js" are set to not display on mobile so the header can fit on smartphone screens. Media queries code is included at the bottom of the "headerbar-style.css". The breakpoint is set at 482 pixels. At this point the header bar will hide any icons that have the class "hbar-hide" on the div in the "headerbar-promo.js". You can change the 482 to any number to set your own breakpoint. Edit the 482 in 4 total places.


MORE OPTIONS & BAR HEIGHT:
See Sticky Header Promo Bar Plugin.



TROUBLESHOOTING
If you are using the global option, and your template uses jQuery includes (like mega menu templates) the header bar HTML code shown in step #4 needs to be added in a different way. See Sticky Header Bar Support.

If you do the steps above and header bar is not working:
  1. Try doing the above steps again from the start.
  2. You are testing the wrong HTML page that you added the header bar code to.
  3. You are editing on-line files instead of files on your hard drive.
  4. You have not used ALL the code from the above steps. Redo steps above.
  5. If you have uploaded: Remember that hosting areas are case sensitive. Be sure all file, folder names and link names match exactly.
  6. Note: Always make backup files and test after each edit before you go to another edit. You can always delete your backups later.
